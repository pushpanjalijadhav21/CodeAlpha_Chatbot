# 🤖 Basic Python ChatBot

A simple rule-based chatbot built using Python. This project was created as part of a Python internship to demonstrate the use of basic programming concepts such as functions, loops, conditional statements, and user input.

## 🚀 Features

- 👋 Greets the user
- 💬 Responds to predefined questions
- 📅 Displays the current date
- 🕒 Displays the current time
- 🧮 Built-in basic calculator
- 📖 Help menu to view available commands
- 👋 Exit command to end the chat

## 🛠️ Technologies Used

- Python 3
- datetime module

## 📚 Python Concepts Used

- Functions
- While Loop
- If-Elif-Else Statements
- User Input & Output
- Exception Handling
- Python Modules (datetime)
